# FAHH

> Someone died. Again.

You know that feeling. You're in a dungeon. Everything's going fine. Then someone stands in the fire. Or the tank pulls wall-to-wall with no cooldowns. Or the healer is alt-tabbed watching YouTube.

And then... **FAHHHHHH.**

## What it does

Every time a group member dies, FAHH plays a random "FAHH" sound. That's it. That's the addon.

- 5 hand-curated FAHH sound variants
- Random selection because variety is the spice of suffering
- Works in dungeons and raids
- Toggle it off if you're a coward

## Commands

| Command | What it does |
|---------|-------------|
| `/fahh` | Toggle on/off |
| `/fahh status` | Show settings |
| `/fahh dungeon` | Toggle for dungeons |
| `/fahh raid` | Toggle for raids |
| `/fahh scenario` | Toggle for scenarios |
| `/fahh world` | Toggle for open world (off by default, you psycho) |
| `/fahh test` | Preview a random FAHH |

## FAQ

**Q: Can I add my own sounds?**
A: Yes. Drop `.ogg` files in the `Sounds/` folder and add them to the `SOUNDS` table in `Core.lua`. Go wild.

**Q: Does it play when I die?**
A: Yes. You deserve it.

**Q: My group is mad at me.**
A: Working as intended.

**Q: Does it work in LFR?**
A: Yes, but you'll need to turn your volume down. Trust me.
